## 二、产品背景与问题陈述

### 背景
小学生每天有固定作业任务，家长需要监督孩子按时完成。核心痛点：
- 家长靠记忆或纸质记录，容易遗漏作业任务
- 孩子缺乏完成作业的正向反馈，难以形成稳定习惯
- 家长无法直观了解孩子的作业完成情况和进步趋势

### 产品定位
家庭自用工具，家长和孩子共用同一套 Web 系统，通过任务设定→打卡→统计的闭环，防止漏写作业，逐步培养孩子自主完成作业的习惯。

### 核心问题
**如何通过低门槛的打卡机制，让孩子愿意坚持、家长省心跟踪？**

## 三、目标用户

| 角色 | 核心需求 |
|------|----------|
| 家长 + 孩子（共用） | 家长负责设定任务和查看统计；孩子负责执行打卡操作 |
| 使用特点 | 同一家庭账号，家长端功能更完整，孩子端界面更简洁、操作更友好 |

---

## 四、产品目标与成功指标

### 产品目标
1. 杜绝漏写作业的情况（任务设定后有提醒，不遗漏）
2. 通过连续打卡、成就反馈等机制，培养孩子每天主动完成作业的习惯
3. 家长零成本掌握孩子的作业完成情况

### 成功指标（上线后衡量）
- 任务设定后按时打卡率 ≥ 80%
- 连续打卡 7 天以上的孩子占比 ≥ 40%
- 家长每周主动查看统计 ≥ 3 次

## 五、核心功能需求

---

### 5.1 任务设定

**功能描述**：家长为孩子的每日/每周作业创建打卡任务。

#### 功能点

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 创建任务 | 支持手动创建，填写任务名称、科目、预计时长、截止时间 | P0 |
| 任务模板 | 预设常见科目模板（语文、数学、英语等），一键套用 | P1 |
| 周期性任务 | 支持按天/周重复设置（如"每天数学口算10分钟"） | P0 |
| 任务编辑 | 修改任务内容、删除任务 | P0 |
| 任务排序 | 家长可调整任务显示顺序，帮助孩子按顺序完成 | P2 |
| 批量导入 | 支持从老师布置的作业图片识别导入（OCR） | P2 |

#### 交互说明
- 创建任务流程不超过 3 步
- 任务名称支持 emoji，增加趣味性（如 📖 语文、🔢 数学）

---

### 5.2 打卡功能

**功能描述**：孩子或家长标记任务完成，记录完成时间和质量。

#### 功能点

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 一键打卡 | 任务完成后点击「完成」按钮，记录完成时间 | P0 |
| 拍照/上传证明 | 支持上传作业照片作为完成凭证 | P1 |
| 完成质量标记 | 孩子/家长可标记完成质量（⭐️⭐️⭐️ 三档） | P1 |
| 打卡提醒 | 到达设定时间推送提醒通知 | P0 |
| 补打卡 | 支持补记前一天未完成的打卡 | P1 |
| 打卡日历 | 展示当月打卡情况，已完成日期高亮显示 | P1 |
| 连续打卡 streak | 显示连续完成天数，激励孩子保持习惯 | P1 |

#### 交互说明
- 打卡操作需在 2 次点击内完成（降低孩子操作门槛）
- 打卡成功后有动画反馈（如 🎉 撒花效果）

---

### 5.3 统计功能

**功能描述**：家长查看孩子的作业完成数据，发现问题并调整策略。

#### 功能点

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 完成率统计 | 按日/周/月展示任务完成率（百分比 + 趋势图） | P0 |
| 科目维度统计 | 各科目完成情况的对比分析 | P1 |
| 完成时长分析 | 实际完成时长 vs 预计时长的对比 | P2 |
| 未完成任务列表 | 展示遗漏的任务及频次 | P0 |
| 打卡日历总览 | 月度日历视图，一目了然查看打卡情况 | P1 |
| 导出报告 | 支持将统计数据导出为图片或 PDF，分享给老师 | P2 |
| 周报/月报 | 自动生成学习报告，总结本周/本月表现 | P1 |

#### 数据指标定义
- **完成率** = 已打卡任务数 / 应打卡任务总数 × 100%
- **连续打卡天数**：连续每天至少完成一项任务的累计天数
- **平均完成时间**：任务截止前 X 小时完成（评估拖延情况）

## 七、非功能性需求

| 类别 | 要求 |
|------|------|
| 性能 | 打卡操作响应时间 < 500ms |
| 可用性 | 孩子（6岁+）可独立操作核心打卡流程，界面字体≥16px |
| 响应式适配 | 同一 Web 应用，自适应桌面（>1024px）、平板（768-1024px）、手机（<768px）三种屏幕 |
| 数据安全 | 作业照片仅存储在家长授权云端，符合未成年人数据保护要求 |
| 离线能力 | 无网络时可打卡，联网后自动同步 |
| 浏览器兼容 | Chrome / Safari / Edge 最新两个版本 |
